Claude 功能全景 · V1.0
大家好，这份 PPT 按照官网左侧导航的章节顺序组织，共 11 节内容。我们从基础功能讲起，重点介绍三大核心工具，最后总结联动工作流与最佳实践。

01New Chat · 新建聊天

创建全新对话，每次都是「从零开始」

独立上下文环境，不受之前对话影响

💡

使用场景

切换话题 · 重置过长上下文 · 多任务并行 · 测试不同 Prompt

⚙️

使用技巧

每任务开新聊天 · 标题注明主题 · 开始时说明背景信息

⚠ 避免：一个聊天混杂太多不相关话题 · 对话过长不清理

01 · new-chat
New Chat 是最基础的功能。适合切换话题、重置上下文、并行处理多个项目。记得给聊天起好标题。

02Search · 搜索

全文搜索历史对话，打造个人知识库

快速定位曾经讨论过的内容、代码与方案

📌

典型用途

找回代码片段 · 查找技术决策 · 复用 API 处理方案 · 知识管理

🎯

搜索策略

用具体关键词（如 React useEffect）· 结合日期范围 · 搜特定格式如 def

React useEffectdocker compose技术方案选择
02 · search
Search 让对话变成可检索的知识库。使用具体关键词，而不是泛泛搜「代码」。

03Chats · 聊天历史

查看与管理所有对话记录

重命名、归档、删除 — 延续未完成任务

💡 使用场景

  • 项目追溯与上下文延续
  • 清理无用、整理重要对话
  • 复习学习记录

📝 命名规范

  • 2024-Q1-财报分析
  • 【重要】客户A需求
  • 产品PRD-v2.0讨论
03 · chats
Chats 是对话管理中心。好的命名习惯让 Search 更高效。
📁
★★★ 核心生产力
