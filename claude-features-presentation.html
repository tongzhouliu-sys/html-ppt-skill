Feature 04 · Projects

Projects — 专属工作空间

知识库 + 自定义指令 + 跨会话记忆，一次配置、持续复用

📄

独立知识库

上传 PDF、代码、图片等多格式文件，Claude 自动理解项目背景

⚙️

自定义指令

设置角色与规范，项目内所有对话自动应用，无需重复说明

🧠

上下文记忆

跨会话保持项目背景，对话历史关联在一起，持续积累

💡 典型场景

  • 软件开发：上传 API 规范 + 需求文档
  • 学术研究：参考文献 + 实验数据 CSV
  • 企业知识库：产品手册 + FAQ 文档

⚡ 使用技巧

  • 先上传核心文档，再补充参考资料
  • 定期更新文件，删除过期内容
  • 为项目设置明确的角色指令
projects · workspace
第一个核心功能是 Projects。你可以把它理解成 Claude 的工作空间——每个项目有自己的知识库、自定义指令和对话历史。比如做软件开发时，把 API 规范和需求文档都传上去，Claude 就能直接参考这些文档来回答。关键是设置好自定义指令，比如"你是资深全栈工程师"，这样每次对话都不用重复背景了。

★★★ 可视化交互

Feature 05 · Artifacts

Artifacts — 即时可视化产物

独立窗口实时预览 · 即时修改 · 一键下载导出

🌐

网页应用

生成可交互 HTML/CSS/JS，右侧实时预览，立即测试功能

待办 App 登录页
📊

数据图表

D3.js / Chart.js 交互式可视化，悬停查看、缩放筛选

趋势图 热力图
📐

SVG 图形

流程图、架构图、示意图，即时渲染可下载

📄

结构化文档

Markdown、API 文档、技术规范，格式化渲染展示

迭代开发： 创建基础版 → 功能扩展 → 交互增强 → 视觉升级，每次修改立即看到效果

artifacts · live-preview
第二个是 Artifacts，这是可视化交互的神器。Claude 生成的网页、图表、文档会在独立窗口里实时预览。你可以说"创建一个待办应用"，然后直接在右侧测试。不满意就继续改——加功能、换配色，每次修改都是即时生效的。最后还能一键下载完整 HTML 或部署到 Vercel。
⚡
★★★ 终端编程助手

Feature 06 · Claude Code

Code — 终端级 AI 编程

直接操作本地文件 · 执行 Shell · 跨文件修改 · 自动测试部署

🔧

代码库重构

扫描架构 → 跨文件修改 → 运行测试 → 自动 git commit

🐛

Bug 修复

执行测试 → 分析日志 → 定位修复 → 验证通过

🚀

DevOps 自动化

Docker 容器化 → CI/CD 配置 → 部署脚本一键生成

claude · terminal
$ claude /review src/auth.ts
→ 自动审查并提出改进建议

$ claude "REST 改为 GraphQL"
→ 搜索文件 → 修改 schema → 运行测试

$ claude "配置 Docker + CI/CD"
→ Dockerfile + compose + GitHub Actions

高级能力

/

斜杠命令

/review · /test · /deploy 快捷操作

🪝

Hooks 钩子

文件变更自动格式化 + 运行测试

🔗

MCP 集成

GitHub · Slack · Jira · 数据库直连

code · terminal-agent
第三个是 Claude Code，运行在终端里的 AI 编程助手。它不只是生成代码——能直接操作你的文件系统、跑 shell 命令、跨文件修改。比如你说"把 REST 改成 GraphQL"，它会搜索相关文件、改 schema、跑测试。还支持斜杠命令、Hooks 自动化，以及 MCP 协议连接 GitHub、Jira 等外部工具。
