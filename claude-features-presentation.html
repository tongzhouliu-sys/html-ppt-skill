基础功能

01 · New Chat

新建聊天 — 从零开始

创建全新对话会话，独立上下文，不受之前对话影响

💡

使用场景

切换话题 · 重置上下文 · 多任务并行 · 测试不同 Prompt

⚙️

使用技巧

每任务开新聊天 · 标题注明主题 · 开始时说明背景

⚠ 避免：一个聊天混杂太多话题 · 对话过长不清理

01 · new-chat
New Chat 是最基础的功能——每次新建对话都是一张白纸。适合切换话题、重置过长的上下文，或者并行处理多个不相关项目。记得给聊天起个好标题，比如「Python 数据分析-客户画像」。

02 · Search

搜索 — 个人知识库

全文搜索历史对话，快速定位代码、方案与决策记录

📌

典型用途

找回代码片段 · 查找技术方案 · 复用 API 处理方案

🎯

搜索策略

用具体关键词 · 结合日期范围 · 搜格式如 def

React useEffectdocker compose技术方案选择
02 · search
Search 让你把和 Claude 的所有对话变成可检索的知识库。比如搜「React useEffect」找之前的代码，或搜「技术方案选择」找回决策记录。关键是使用具体关键词，而不是泛泛的「代码」。

03 · Chats

聊天历史 — 对话管理

查看、重命名、归档、删除所有对话，延续未完成任务

💡 使用场景

  • 项目追溯与上下文延续
  • 清理无用、整理重要对话
  • 复习学习记录

📝 命名规范

  • 2024-Q1-财报分析
  • 【重要】客户A需求
  • 产品PRD-v2.0讨论
03 · chats
Chats 是对话管理中心。定期重命名、加前缀标记，按项目分组。好的命名习惯能让 Search 更高效——这是知识管理的基础。
